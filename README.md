# embed-gist

Embed GitHub Gists in [Inkdrop](https://www.inkdrop.app/) notes.

Paste a Gist link and it renders as an embedded code snippet in the preview. A paste-link format option lets you quickly insert the iframe HTML directly in the editor.

## Features

- Renders GitHub Gists inline in the Markdown preview
- Supports `gist.github.com/...` URLs
- Adds a "Gist embed" option to the paste-link autocomplete menu for inserting iframe HTML

## Install

```
ipm install embed-gist
```

## Usage

1. Paste a GitHub Gist URL on its own line in a note
2. The preview will render it as an embedded code snippet
3. When pasting a Gist URL, select **Gist embed** from the autocomplete menu to insert the iframe HTML directly
